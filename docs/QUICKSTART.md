# Quick Start Guide

Welcome to the TulparLang Quick Start Guide! This document will help you get started with TulparLang in no time.

## Installation

Follow the instructions in the main README to install TulparLang on your system. Once installed, you can start writing and running Tulpar programs.

## Your First Program

Create a file named `hello.tpr` with the following content:

```tulpar
str message = "Hello, World!";
print(message);
```

Run the program using the Tulpar interpreter:

```bash
./tulpar hello.tpr
```

You should see the following output:

```
Hello, World!
```
