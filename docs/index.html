TulparLang

Modern, statik tipli betik dili. LLVM tabanlı AOT derleyici.

Kurulum

Windows (PowerShell):

iwr -useb https://tulparlang.dev/install.ps1 | iex

Linux / macOS:

curl -fsSL https://tulparlang.dev/install.sh | bash

Yeniden çalıştırmak en son sürüme yükseltir. Kurulu sürümden sonra:

tulpar update --check

Hızlı başlangıç

// merhaba.tpr
print("Merhaba dünya");

tulpar merhaba.tpr
